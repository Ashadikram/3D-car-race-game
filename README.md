A simple and fast-paced 3D Car Racing Game built using Unity. This project includes smooth car controls, basic physics, UI elements, and a complete racing track. Perfect for beginners who want to learn Unity game development.

## Features
- Smooth car movement and physics
- 3D racing track with obstacles
- Lap timer and basic UI

## Technologies Used
- Unity (C#)
- Basic 3D models
- Unity UI System

## How to Play
- Use **WASD / Arrow Keys** to control the car
- Follow the track and complete laps

## How to Run
1. Clone or download the project
2. Open the project in **Unity**
3. Load the main scene
4. Press **Play** to start the game
